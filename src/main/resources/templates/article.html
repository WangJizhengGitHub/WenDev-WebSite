2020-1-1 Spring Cloud Java 300

一级标题 h1

启动Nacos 官网：https://nacos.io/zh-cn/ 下载地址：https://github.com/alibaba/nacos/releases 解压后使用以下命令启动： 

sh nacos/bin/startup.sh -m standalone

其中standalone表示以单机模式运行，具体可以参考文档。 启动后访问127.0.0.1:8848，默认用户名和密码都是nacos，登录后就可以看到Nacos控制台了。
